# yt2alt

yt2alt is a command line tool to export your profile from Youtube and import it to an alternative platform, such as Invidious.

## Usage

```bash
npx yt2alt
```
## Features

Reads from Youtube:
- Subscriptions
- Watch history
- Liked videos
- Watch later
- Recommended videos
- Playlists

Writes to
- Invidious (Import API)
- Invidious (save to file)
